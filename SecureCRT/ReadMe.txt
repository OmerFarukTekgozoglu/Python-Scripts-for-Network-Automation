Hello,

This script is written for getting CFCARD Memory usage of Huawei Routers. We had use this script for an upgrade operation information collection process.

Script is working with SecureCRT Script-Runner. 

Device List .csv file includes the device ip addresses and other releated informations.

For running this script; you should have permission to use an NMS Linux device CLI screen with SecureCRT. You should login your account with using CLI (NMS Server).  Then run the script via SecureCRT Script Runner.
